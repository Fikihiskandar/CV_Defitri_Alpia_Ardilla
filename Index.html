Ringkasan Profesional

Saya merupakan lulusan dari Universitas Pamulang, Tangerang Selatan jurusan Manajemen Keuangan. Dalam studi saya belajar mengenai laporan keuangan seperti laporan laba rugi, laporan arus cash, neraca, laporan perubahan ekuitas, catatan atas laporan keuangan, serta rasio - rasio keuangan.

Saya memiliki pengalaman kerja selama kurang lebih enam tahun di industri retail dan distribusi, khususnya dalam manajemen inventaris dan pengelolaan aset. Saat ini, saya menjabat sebagai Inventory Specialist di PT Indomarco Prismatama (Indomaret), dengan tanggung jawab utama memastikan data inventaris gudang, pelaporan aset, pumusnahan AT fisik dan non fisik, serta efisiensi proses administrasi melalui sistem berbasis Oracle.

Keahlian saya mencakup pengelolaan data inventaris dan basis data seperti MySQL, PHPMyAdmin, dan Oracle. Saya juga memiliki kemampuan dalam pengolahan data menggunakan Microsoft Excel, Microsoft Word, Microsoft Power Point serta analisis data untuk mendukung pengambilan keputusan.
